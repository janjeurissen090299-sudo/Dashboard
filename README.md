# Jan's Dashboard

Persoonlijk dashboard — training, slaap, stretch, MMA mindset.

## Live zetten via GitHub Pages (gratis)

### Stap 1 — GitHub repo aanmaken
1. Ga naar [github.com](https://github.com) en log in
2. Klik rechts bovenaan op **+** → **New repository**
3. Naam: `dashboard`
4. Zet op **Public**
5. Klik **Create repository**

### Stap 2 — Bestanden uploaden
1. In je nieuwe repo, klik op **uploading an existing file**
2. Sleep deze 3 bestanden erin: `index.html`, `manifest.json`, `README.md`
3. Klik **Commit changes**

### Stap 3 — GitHub Pages aanzetten
1. Ga naar **Settings** (tabblad bovenaan je repo)
2. Links in het menu: **Pages**
3. Onder "Branch": selecteer **main** → **/root** → klik **Save**
4. Na ~1 minuut is je dashboard live op: `https://JOUWGEBRUIKERSNAAM.github.io/dashboard`

### Stap 4 — Installeren als app op je telefoon

**iPhone (Safari):**
1. Open de URL in Safari
2. Tik op het **deelknopje** (vierkantje met pijltje omhoog)
3. Kies **Zet op beginscherm**

**Android (Chrome):**
1. Open de URL in Chrome
2. Tik op de **drie puntjes** rechtsboven
3. Kies **Toevoegen aan beginscherm**

## Icoontjes toevoegen (optioneel)
Maak twee PNG icoontjes (192×192 en 512×512) via [realfavicongenerator.net](https://realfavicongenerator.net) en upload ze als `icon-192.png` en `icon-512.png` in je repo.
